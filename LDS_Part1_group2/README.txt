Descrizione cartelle:
- Codici creazione tabelle: contiene gli script per la creazione delle tabelle, ma senza correzione di missing values o errori.

- Codici gestione missing values: contiene gli script per pulire i file csv e riempire/eliminare missing values, in modo da produrre le tabelle da inserire.

- Codici inserimento: contiene gli script per l'inserimento delle tabelle nel DB.

- Datasets: contiene ogni file csv usato in questa parte del progetto (dai file iniziali grezzi senza i file tennis.csv, female_players.csv, male_players.csv e countries.csv, ai file finali puliti e pronti all'inserimento)
